# Fingerprint Attendance System using ESP32

## Features
- Fingerprint Authentication
- Subject-wise Attendance
- Google Sheets Integration
- Wi-Fi Enabled
- LED & Buzzer Indication
- Teacher Authentication

## Components
- ESP32
- R307 Fingerprint Sensor
- I2C LCD
- Buzzer
- LEDs

## Software
- Arduino IDE
- Google Apps Script
- Google Sheets

## Working
Teacher activates attendance using fingerprint. Students scan fingerprints and attendance is stored automatically in Google Sheets.
